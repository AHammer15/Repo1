Git

Is local

Is a version control system

Repository - The location where your code history is stored

Branch - "Folder" within the repository containing commits

Commits - Different versions of your code within a branch

Command

$git init

Initiazes a local git repository

$git status

Shows status, information about current state

$git add .

Adds all files in your project folder into your repository, tracks changes in all files inside the repository

$git commit -m

Saves current version of code in branch and gives it a name (name added as "name" after -m)

$git branch

Lists all branches in repository

$git log

Lists all commits inside branch

$git reset --hard "commit ID"

Reverts to a previous commit

$git checkout -- .

Reverts unstaged changes and goes back to last commit

$git checkout -b name

creates new branch with "name" as the name

$git merge name

Merges currently selected branch with branch "name"

$git branch -D name

Deletes branch "name"

GitHub

Is web based

Is a hosting & collaboration provider

Command

$git push

Allows us to upload the contents of our local repository to the remote one
